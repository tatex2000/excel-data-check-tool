# Excel データチェック＆修正候補抽出ツール（VBA）

Excelデータの「表記の違い」や「入力ゆれ」を検出し、  
修正候補または要確認として一覧出力するツールです。

---

## ■ このツールでできること

- データの不一致・入力ゆれを自動検出
- 修正候補を一覧表示（そのままコピーして修正に使える）
- 要確認データも同時に抽出
- ヘッダ名ベースで汎用的に判定（業種問わず使用可能）

---

## ■ 特徴

- VBAのみで動作（追加インストール不要）
- ルールはシート入力だけで変更可能（コード修正不要）
- 任意のExcelデータに対応（名簿／商品マスタ／顧客データなど）

---

## ■ 使用イメージ

例：

| 項目 | 元データ | 修正候補 |
|------|--------|----------|
| 氏名 | " 山田太郎 " | "山田太郎" |
| 住所 | "東京都　千代田区" | "東京都 千代田区" |

→ 修正候補をそのままコピーして利用可能

---

## ■ 主な判定ルール

- 含む / 含まない
- 完全一致 / 前方一致 / 後方一致
- 前後空白あり
- 全角スペースを含む
- 半角スペースを含む
- 連続スペースあり
- 英字 / 数字 / 記号を含む
- 英字＋日本語混在
- 空白 / 空白でない

※すべてプルダウンで選択可能

---

## ■ ファイル構成

- `Excel_Data_Check_Tool.xlsm` … メインツール
- `sample_master_1.xlsx` … サンプル（名簿系）
- `sample_master_2.xlsx` … サンプル（商品データ）
- `sample_master_3.xlsx` … サンプル（混在データ）

---

## ■ 使い方（簡易）

1. 「データ取込」ボタンでファイルを選択  
2. ルール入力シートで判定ルールを設定  
3. 「判定実行」  
4. 判定一覧・最終判定を確認  

---

## ■ 注意事項

- 「候補」を選択したルールのみ、最終結果に反映されます
- 修正は自動反映ではなく、コピーして使用する設計です
